# 📊 Customer Intelligence Dashboard

## 🚀 Overview
Built an end-to-end Customer Behavior Analysis system to understand purchasing patterns, customer segments, and business insights using real-world e-commerce data.

---

## 🎯 Problem Statement
Businesses often struggle to understand:
- Who their most valuable customers are  
- What drives purchasing behavior  
- How discounts and trends affect revenue  

---

## 💡 Solution
Developed an interactive dashboard that:
- Analyzes customer demographics  
- Tracks spending behavior  
- Evaluates discount effectiveness  
- Provides actionable business insights  

---

## ⚙️ Tech Stack
- Python  
- Pandas  
- Plotly  
- Streamlit  

---

## 📊 Features
- 📈 Interactive dashboard  
- 🎯 Smart filters (Gender, Category)  
- 💰 KPI metrics (Revenue, Avg Purchase)  
- 📊 Visual insights (Charts & Trends)  
- 📥 Downloadable filtered data

## 📈 Key Insights
- Customers aged 25–40 contribute highest revenue  
- Discounts positively influence purchase behavior  
- Frequent buyers drive consistent sales  
- Seasonal trends impact overall revenue
